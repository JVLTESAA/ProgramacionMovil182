¿Qué es JS?

Fue un lenguaje que se escribió para los exploradores web, por lo que todos los exploradores en la actualidad cuentan con un motor que se encarga de ejecutar JS (JavaScript Engine)
Lenguajes de programación mas populares de todo el mundo, con el cual se puede trabajar en el desarrollo web, así como en desarrollo Frontend que se encarga de construir toda la parte visual de la aplicación, así también del desarrollo Backend que se encarga de la programación en la parte del servidor, desarrollador Fullstack que se puede encargar del Frontend y del Backend.
Ademas se puede tabajar como desarrollador movile tanto como para Android y iOS, también se pueden construir aplicaciones de escritorio.

Por ultimo se puede trabajar con microcontroladores.

Frameworks: entrega muchas herramientas para desarrollar aplicaciones complejas y te instruye en como trabajar sobre ellas. El Framework entrega mas herramientas que una librería.

Librería: Se encarga en solucionar un problema en especifico, sin embargo también es una herramienta que solo se encarga en resolver un problema en especifico

 ¿Qué es ECMAScript?

European Computer Manofacturers Association, el cual a contribuido con varios estándares técnologicos como:
ECMA 334 -> Especificaciones técnicas de C#
ECMA 404 -> Especificaciones técnicas de JSON
ECMA 408 -> Especificaciones técnicas de Dart

ECMA 262 O ECMAScript -> El cual indica los estándares para los lenguajes de programación JavaScript, JSript y de ActionScript.

ECMAScript 2015 -> Actualización de versión que añadieron funcionalidades como:
